# smol-telaproduto-eduardo-borges

Foi um teste desafiador, bem objetivo e que testa os principais conhecimentos em programação e
tecnologias como seguir o design como na tela dos bombons, lógica de programação para fazer a soma
de valores dos ingredientes ao valor final do bombom podendo adicionar mais bombons, a integração da
API com os 4 métodos GET, POST, PUT e DELETE. Tive dificuldade com a passagem da data de nascimento
com GET para mostrar as datas nos formatos dd/mm/yyyy que estavam em yyyy-mm-dd principalmente para
dar PUT.
Gostei muito do teste que colocou em pratica alguns conhecimentos meus e agradeço muito pela
oportunidade de estar participando deste processo.
Aproximadamente foi trabalhado 28 horas para completar.
